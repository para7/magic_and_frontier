# 導入方法(開発版)

まるごと zip でダウンロードしてから、解凍します。

入ってるものを全部導入してください。

以下のコマンドを実行して初期化します

```
/function maf:magic/dev
/function maf:system/set_player_id/run
```

過去バージョンから更新する場合、次のコマンドで更新可能です。

```
/function maf:reinstall
```
